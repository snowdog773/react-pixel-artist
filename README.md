# React Pixel Artist

This is the front end to my 20 Squared pixel artist project which was one of my major coursework pieces on my bootcamp which ended in May 2022. This front end uses React to create a grid of 20 x 20 "pixels" which users can colour from a pallette to create pieces of pixel artwork. Creating an account takes seconds and doesn't require any personal details, please take the time to sign up and create some artwork on the following link.

<https://20squared.pitans.co.uk/>
